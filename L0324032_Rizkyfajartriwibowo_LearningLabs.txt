# Website Kuliner Nusantara

Website Kuliner Nusantara adalah platform digital yang dirancang untuk memperkenalkan dan mengeksplorasi kekayaan kuliner Indonesia. Website ini menyajikan informasi tentang berbagai makanan tradisional dari berbagai daerah di Indonesia.

## Fitur Utama

### 1. Halaman Utama (index.html)
- Hero section dengan gambar latar belakang kuliner
- Highlight kuliner unggulan (Rendang, Soto, Nasi Goreng)
- Animasi hover pada kartu kuliner
- Navigasi yang responsif
- Header dengan efek scroll

### 2. Halaman Eksplorasi (content.html)
- Katalog makanan tradisional Indonesia
- Informasi detail tentang setiap makanan
- Fitur "Load More" untuk memuat konten tambahan
- Deskripsi lengkap bahan dan variasi makanan
- Gambar ilustrasi untuk setiap makanan

### 3. Halaman Feedback (feedback.html)
- Form feedback interaktif
- Sistem rating bintang
- Validasi input form
- Notifikasi sukses setelah submit

## Teknologi yang Digunakan

1. **Framework CSS**
   - Tailwind CSS untuk styling responsive
   - Sistem grid modern
   - Komponen yang dapat digunakan kembali

2. **JavaScript**
   - Interaksi dinamis pada form
   - Animasi scroll header
   - Sistem rating interaktif
   - Loading konten dinamis

3. **HTML5**
   - Semantic markup (header, nav, main, footer)
   - Form validation
   - Responsive images

Website ini dirancang untuk responsif di berbagai ukuran layar:
- Desktop (> 1024px)
- Tablet (768px - 1024px)
- Mobile (< 768px)
